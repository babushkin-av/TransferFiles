(all options)
Options Description/TH>
-h, --help Show help-screen and exits.
-V, --version Show version info and exits.
-q, --quiet Suppress informational messages. However, error reports will be displayed.
-d, --debug Show debug messages. This supersedes '-q' and/or '--quiet' options.
-f, --force Try to recover from errors.
-r, --recursive Dive into directories recursively.
-n --nofollow Do not follow symbolic links.
-o --overwrite Overwrite existing files.
-4, --ipv4, This is a hint as to which Protocol to use when you specify a host name.
-T=[addr], --to=[addr] Work as a client sending files TO [addr] (which is localhost by default).
-F=[addr], --from=[addr] Work as a server reciving files FROM [addr] (which is INADDR_ANY by default).
-p=[port], --port=[port] Set up a port number (which is 45678 by default).
